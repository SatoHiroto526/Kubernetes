# discovery-lbオブジェクト
## Service
```
クラスター上に展開したPodにアクセスするためのエンドポイントを提供するオブジェクト。
作成されるとクラスター内で一意なIPアドレスが割り振られ、クラスター内ではそのIPアドレスでアクセス可能。
Serviceのspec.selectorで指定されたラベルを持つPodにルーティングされる。
また、ラベルセレクタを使用せず、クラスターに外部に接続するエンドポイントを作成することも可能。
L4ロードバランサーとして機能。
バックエンドのPodの増減に合わせて配送先を追加、削除する。
```

## Ingress
```
クラスター外部からのリクエストをServiceに配送するオブジェクト。
HTTP／HTTPSのみ受け付ける。（Serviceはあらゆるプロトコルを受け付ける。）
1つのIngressから複数のServiceに配送できる。（Serviceは1つにつき1つのPodラベルしか扱えない。）
L7ロードバランサーとして機能。
```
