# workloadsオブジェクト
```
Kubernetesクラスタ上にコンテナを起動したり、ワークロードの要件に合わせて維持、管理するためのオブジェクト
```

## workloadsオブジェクトの種類
### Pod
```
コンテナ管理の最小単位のオブジェクト。
全てのコンテナはPodに含まれ、管理・実行される。
```

### ReplicaSet
```
Podの数を維持したり、スケールアウト・インするためのオブジェクト。
```

### Deployment
```
アプリケーションの管理を安全に実行するための管理をするオブジェクト。
```

## workloadsオブジェクトの親子関係
```
（親）Deployment ⇒ ReplicaSet ⇒ Pod（子）
```
